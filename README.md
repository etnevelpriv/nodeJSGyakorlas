A mellékletben található feladatsort készítsd el NodeJS segítségével.

Hogyan redszerezd a forrásfájlokat?
Három pojektet készíts!
A 2. és 3. feladatnak legyen külön belépési pontja (index.js)
A 2. feladat függvényeit egy külön "feladat2.js" fájlban helyezd el
A 3. feladat osztályát egy megfelelően elnevezett fájlba helyezd el. Ha a megoldás során használtál segédfüggvényeket, azokat is ebbe a fájlba helyezd.

A konzolról olvasáshoz, ill. a fájlműveletekhez használhatod segítségül a korábbi projekteket.
